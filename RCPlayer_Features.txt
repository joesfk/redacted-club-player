===============================================================================
                    RCPlayer — Complete Feature List
              Retro Terminal Music Player with ARG Mechanics
===============================================================================

OVERVIEW
  Single-file HTML app (~5000 lines). Vanilla JS, CSS3, HTML5.
  No frameworks. No build system. No external dependencies (except Google Fonts).
  Themed as a cyberpunk hacker terminal with puzzle/ARG mechanics.

===============================================================================
1. POPUP WINDOWS (13 total)
===============================================================================

  PLAYER.EXE      Audio player — track list, play/pause/skip, volume, download
  GALLERY.EXE     Image gallery — 6 unlockable images, lightbox viewer
  SCANNER.EXE     Frequency scanner — dual analog knobs, spectrum viz, signal lock
  RADAR.EXE       Circular radar — rotating sweep, ping waves, echo detection
  ECHOES.EXE      Echo log — detected signals, proximity %, lock status
  SNAKE.EXE       Classic snake game — WASD/arrows, high score (localStorage)
  TRON.EXE        Lightcycle game vs AI — collision, win tracking (localStorage)
  CALC.EXE        Calculator (uses eval for retro feel)
  NOTEPAD.EXE     Editable text notepad
  MATRIX.EXE      Matrix rain animation (canvas)
  DONUT.EXE       Spinning 3D ASCII donut (rotating torus)
  PINCODE.EXE     4-digit PIN pad — codes: 1337, 9775, 1984, 0420, 0000
  SYNTH.EXE       Keyboard synthesizer (see section 8)
  DRUMS.EXE       Drum machine (see section 8)

2. TERMINAL COMMANDS (50+)
===============================================================================

  CORE
    help            Show command list (tier-gated)
    clear           Clear terminal output
    status          System info (tier, format, volume, secrets)
    list            View available tracks with lock/unlock status
    echo [text]     Echo text (special handling for "redacted")

  FILE SYSTEM
    files / ls      List directory contents
    read / cat      Read files (manifest.log, system.log, frequencies.log)
    search / grep   Search through files

  AUDIO PLAYBACK
    player / music  Open PLAYER.EXE
    play [n]        Play track by number
    pause           Pause playback
    stop            Stop playback
    next / prev     Skip tracks
    vol [0-100]     Set volume
    format          Toggle MP3/WAV

  SIGNAL / SCANNING (Tier 2+)
    scan            Open SCANNER.EXE
    tune [freq]     Lock onto frequency
    decode [key]    Decrypt locked tracks
    signal          Shortcut to unlock RC-005
    beneath         Shortcut to unlock RC-006

  SYSTEM (Tier 3+)
    whoami          Identity query
    source          View hidden notes
    secrets         List discovered secrets
    ping            Signal test

  WINDOW LAUNCHERS
    gallery/photos  Open GALLERY.EXE
    calc            Open CALC.EXE
    notepad         Open NOTEPAD.EXE
    snake           Open SNAKE.EXE
    tron/sark/mcp   Open LIGHTCYCLE.EXE
    matrix          Open MATRIX.EXE
    donut           Open DONUT.EXE
    pincode/pin     Open PINCODE.EXE
    synth           Open SYNTH.EXE
    drums           Open DRUMS.EXE

  EFFECTS
    reality/wake    3D zoom-out showing terminal in server rack
    melt            Terminal melt effect
    matrixmelt      Matrix rain melt effect
    screensaver     Flying toasters with glitch corruption

  EASTER EGGS
    redacted        Jurassic Park easter egg (GIF + Nedry sound)
    sfx [on/off]    Sound effects controls

  RESTRICTED (fun responses)
    exit/quit       "There is no exit"
    cd              "restricted"

8. MUSIC PRODUCTION
===============================================================================

  SYNTH.EXE
    4 waveforms         Sine, square, sawtooth, triangle
    2-octave keyboard   14 white + 10 black keys (rendered dynamically)
    Octave shift        -2 to +3 range
    ADSR envelope       Attack, decay, sustain, release sliders
    Low-pass filter     Cutoff (100-20000 Hz) + resonance (0-30)
    Volume              Independent from player
    16-step sequencer   Record notes, loop playback
    Keyboard mapping    A-L = white keys, W E T Y U I O P = black keys
    Web Audio API       Separate AudioContext from player

  DRUMS.EXE
    8 synthesized pads  Kick, snare, hi-hat, open hat, clap, tom, rim, cymbal
    16x8 step grid      Click cells to toggle patterns
    Pad triggering      Click pads or press keys 1-8
    Volume              Independent from synth and player
    Web Audio synthesis  All sounds generated (no samples required)

  SHARED TRANSPORT (syncs SYNTH + DRUMS)
    Unified play/stop   Pressing PLAY on either runs both
    Shared recording    REC arms both — notes go to correct instrument
    Synced BPM          Changing BPM on either updates both (60-200)
    Single tick loop    Both sequencers step together in lockstep
